Cursos y Acreditaciones

Pese a que no he tomado muchos cursos, en muchas ocasiones veo videos mas cortos o leeo documentacion de lenguajes para entender funciones y aprender acerca de algun tema, no cuento con acreditaciones. los cursos que he iniciado son:

Cursos

Desarrollo Web Completo con HTML5, CSS3, JS AJAX PHP y MySQL

En este curso se enseña a crear diseños modernos y funcionales utilizando HTML, CSS y JavaScript, junto con metodologías como Módulos y BEM. También se cubre el desarrollo de sitios web dinámicos con PHP y MYSQL, la creación de aplicaciones CRUD y la integración de pagos de PayPal. Se prepara al estudiante para aplicar a trabajos como Desarrollador Web Junior.

Progreso: 30%

CCNAv7: Introduction to Networks

Curso que proporciona una comprensión fundamental de las redes informáticas. Los estudiantes aprenden sobre conceptos como la arquitectura de red, protocolos de comunicación, direccionamiento IP, modelos de capas, dispositivos de red y seguridad. También se les enseña a configurar y solucionar problemas de redes básicas.

Progreso: 100%

CCNAv7: Switching, Routing, and Wireless Essentials

Es un curso de formación de Cisco que cubre los conceptos básicos de la conmutación y el enrutamiento de redes, así como la tecnología inalámbrica. Los participantes aprenden a configurar y solucionar problemas de switches, routers y puntos de acceso inalámbricos, así como a diseñar y mantener redes de área local (LAN) y redes de área amplia (WAN). También se enseña sobre protocolos de enrutamiento, seguridad de redes y tecnologías de redes inalámbricas.

Progreso: 50%

Curso de JAVASCRIPT desde CERO (Completo) - Nivel JUNIOR

En este curso se explican los fundamentos de JavaScript y se desarrolla la logica del lenguaje (10 horas)

Progreso: 100%
